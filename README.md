# Face Mask Detection Using CNN

## 📌 Project Overview

This project is a Deep Learning-based Face Mask Detection System built using TensorFlow, Keras, OpenCV, and Streamlit. The model classifies whether a person is wearing a face mask or not from an input image.

The system was trained on a dataset containing images of people with and without masks and achieved high classification performance.

---

## 🚀 Features

* Face Mask Detection using Convolutional Neural Networks (CNN)
* Image Upload through Streamlit Web Application
* Real-Time Prediction
* Deep Learning-based Binary Classification
* User-Friendly Interface
* Deployable on Streamlit Cloud

---

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Scikit-Learn
* Streamlit
* Matplotlib
